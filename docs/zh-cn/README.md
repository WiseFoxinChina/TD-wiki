## ❔什么是抽动

### 定义

- 现象学仅探讨如何描述症状，而不试图构建解释性的模型。
- 抽动症状通过现象学描述。
- 美国精神医学学会《精神障碍诊断统计手册》第5版修订版（DSM-5-TR）的定义如下：

> 抽动是突然的、快速的、反复的、非节律性的发声或运动。

- 通俗地说，抽动是骤然发生，持续时间相对比较短，多次出现，没有节律（用以与震颤区别）的声音或动作。

### 流行

- 抽动并不是罕见的情况，相反，抽动在儿童期非常常见。

### 分类

据临床观察，抽动的具体表现复杂而多变。可依靠部位的不同，将抽动分为运动抽动和发声抽动。可根据复杂程度的不一，将抽动分为简单抽动和复杂抽动。

- **简单运动抽动**有眨眼、摇头、点头、歪嘴、皱鼻、收腹等。
- **复杂运动抽动**有挤眉弄眼、眼球转动、四肢甩动、用拳击胸、扭动躯干、走路转圈、秽亵动作等。
- **简单发声抽动**有单音、吸鼻子、吼叫、哼哼声、清嗓子、咳嗽声、尖叫声、犬吠声、鸟鸣声等。
- **复杂发声抽动**有单词、词组、短语、短句、重复单词或短语、重复语句、模仿言语、突然改变音量或声调、秽语等。

#### 秽语、秽行与模仿现象

- 据统计，约10%~19%的抽动秽语综合征患者会出现秽语症状。
- 这些秽亵相关的症状可以表现为，突然、大声地发出涉及性、排泄、亵渎宗教等不合时宜的词句。
- 同样包括秽语行为，精神性秽语等。秽语行为是指以手势代替秽语，精神性秽语是指在头脑内重复某个秽亵词汇。
- 部分抽动表现为模仿现象。
- 其常见的形式是模仿语言、动物声音、电视中的特殊声响等。

### 特点

除了定义中述及的特点，抽动还具有以下特征。

#### 先兆冲动

- 抽动发生前，常会出现一种不愉快的体验，被描述为一种瘙痒、灼烧、紧张或者“想要抽动的冲动”的感觉。
- 当抽动“恰到好处”时，这种感觉才会得到短暂缓解。
- 这种症状也被称为感觉性抽动。

#### 波动性

- 抽动的形式、频率、强度会随时间波动。
- 也就是说，随着时间的发展，旧抽动可能消退，新抽动可能出现。
- 这是抽动的一个典型特点。

#### 可抑制性

- 患者或可自愿地、短暂地抑制抽动症状，但最终往往无法抗拒抽动冲动。

### 问题和澄清

- 抽动不是“坏习惯”，不是道德修养或者意志力问题。
- 打骂不会减轻抽动，不良的家庭环境或者成长经历会加重抽动，并增加其共患病的可能性，如强迫症。
- 抽动并不是结膜炎、咽炎或鼻炎等的症状。
- 患者可能会在抽动后表现得若无其事，或者用托词来缓解尴尬。

## ❓什么是抽动障碍

### 定义

- **抽动障碍是一组以抽动为主要临床表现的神经发育障碍。**

### 流行情况

- **抽动障碍不是少见病、罕见病，而是儿童青少年期的常见病。**
- 国外调查
  - 学龄前儿童抽动障碍的患病率为12%~16%。
    - 短暂性抽动障碍为4%~20%。
    - 慢性抽动障碍为1%~2%。
    - 抽动秽语综合征为0.3%~0.8%。

- 2022年国内儿童流行病学报道
  - 6~16岁人群中抽动障碍总患病率为2.5%。
    - 短暂性抽动障碍为1.2%。
    - 慢性抽动障碍为0.9%。
    - 抽动秽语综合征为0.4%。


### 诊断标准

- 抽动障碍的最新诊断标准是DSM-5-TR和ICD-11。
- 抽动障碍根据病程、临床表现，分为短暂性抽动障碍、慢性运动或发声抽动障碍，以及抽动秽语综合征。
- 目前并无可以直接确诊抽动障碍的指标或者检查。
- 某些检查可能提示一些异常，例如脑电图提示轻度非特异性异常，但是这些异常可能并无临床意义。
- 抽动障碍的诊断依据症状学标准。
- 典型的抽动症状并不难于诊断。

#### 抽动秽语综合征

| **DSM-5-TR中抽动秽语综合征的诊断标准**                       |
| ------------------------------------------------------------ |
| A.多种运动和一个或多个发声抽动在障碍的某段时间内出现，尽管不一定同时出现。 |
| B.抽动的频率有高有低，但这种情况自第一次抽动发生起超过1年。  |
| C.于18岁之前发生。                                           |
| D.这种障碍不能归因于某种物质（如可卡因）的生理效应或其他躯体疾病（如亨廷顿病、病毒后脑炎）。 |

#### 慢性运动或发声抽动障碍

| **DSM-5-TR中持续性（慢性）运动或发声抽动障碍的诊断标准**     |
| ------------------------------------------------------------ |
| A.在病程中出现了单一或多种运动抽动或发声抽动，但运动和发声两者并非同时出现。 |
| B.抽动的频率有高有低，但这种情况自第一次抽动发生起超过1年。  |
| C.于18岁之前发生。                                           |
| D.这种障碍不能归因于某种物质（如可卡因）的生理效应或其他躯体疾病（如亨廷顿病、病毒后脑炎）。 |
| E.从不符合抽动秽语综合征的诊断标准。                         |

#### 暂时性抽动障碍

| **DSM-5-TR中暂时性抽动障碍的诊断标准**                       |
| ------------------------------------------------------------ |
| A.单一或多种运动和/或发声抽动。                              |
| B.这种情况自第一次抽动发生起持续少于1年。                    |
| C.于18岁之前发生。                                           |
| D.这种障碍不能归因于某种物质（如可卡因）的生理效应或其他躯体疾病（如亨廷顿病、病毒后脑炎）。 |
| E.从不符合抽动秽语综合征或持续性（运动）或发声抽动障碍的诊断标准。 |

### 病程和预后

- 有资料表明，在儿童期起病的抽动障碍患者：
  - 在青春期过后40%~50%的患者抽动症状自然缓解。
  - 25%~30%的患者抽动症状明显减轻。
  - 剩下25%~30%的患者抽动症状迁延到成年。

- 影响抽动障碍预后的因素很多：
  - 共患病给增加了抽动障碍的复杂性和治疗难度。
  - 年龄、疾病严重程度、诊断、治疗等因素同样影响抽动障碍的预后。

- 儿童抽动障碍可转化为成年抽动障碍。

#### 误区和澄清

- 不应在“自愈”上豪赌，因为：
  - 不是所有的患者都能通过青春期自然缓解。
  - 不经治疗的抽动障碍对患者的社交形象、心理发展、生活质量、学习效率、社会功能等方面产生难以估量的影响。

- 抽动障碍不影响智力。

### “三元性”

抽动障碍具有三元性。

- **抽动障碍是神经发育障碍。**
  - 神经发育障碍是一组在发育阶段起病的疾病（DSM-5-TR），通常是行为或认知障碍（世界卫生组织《国际疾病分类》第11版，ICD-11）。
  - 神经发育障碍还包括智力障碍、孤独症谱系障碍、注意缺陷多动障碍等。

- 抽动障碍存在较明显的精神障碍特点。
  - 什么是精神障碍（DSM-5）
    - 是以临床显著的个体认知、情感调节或者行为紊乱为特征的一组综合征。
    - 它反应了个体心理、生理、发育过程中相关的精神功能的障碍。
    - 通常与社会、工作或其他重要活动中的重大困扰或功能损害相关。
    - 因为丧失亲人（如爱人死亡）而产生的可以预见的、文化认知所理解的反应，并不属于精神障碍。
    - 一些社会偏差行为（例如由于政治、宗教、性）主要属于个人与社会的冲突，这类问题也不属于精神障碍。除非这些偏差行为或冲突是由个体的上述精神功能障碍所导致的。

  - 抽动障碍的“精神障碍性”
    - 抽动障碍常合并有强迫症、焦虑障碍、抑郁障碍等精神障碍。
    - 抽动与情绪压力有非常密切的联系。
    - 抽动症状的先兆冲动或是一种具有精神症状特点的感觉障碍。
    - 心理治疗对抽动症状和抽动障碍患者的生活困扰都有积极影响。

- 抽动障碍可属于神经系统疾病，但与标准运动障碍性疾病又有所不同。
  - 神经系统疾病：
    - 主要体现为运动、感觉、反射、自主神经以及高级神经活动方面的功能障碍。

  - 运动障碍性疾病：
    - 又称为锥体外系疾病。
    - 是一组以不自主运动、自主运动迟缓、肌张力异常等运动症状为主要表现的神经系统疾病。
    - 包括帕金森综合征、舞蹈病及相关综合征等。
    - 主流观点认为，抽动障碍表现为不自主运动，但运动功能正常，没有清晰准确的器质性病变。


### 抽动障碍的共患病

- 共患病的意思是“一人多病”，且多种疾病难以互相区分主次，缺乏必然因果联系。
- 约80%的抽动秽语综合征患者至少共患有一种精神障碍。
- 约60%的抽动秽语综合征患者共患两种或以上的精神障碍。
- 注意缺陷多动障碍、强迫症都是常见的共患病，在抽动秽语综合征患者中患病率已逾半数。
- 共患病增加了抽动障碍的复杂性和严重性，给个体造成严重的日常生活影响，并给疾病的诊断、治疗和预后增加了更多的困难和挑战。

#### 注意缺陷多动障碍

- 注意缺陷多动障碍（ADHD），是起病于儿童期的常见的神经发育障碍。
- ADHD的症状与抽动障碍大相径庭。
- 其以注意力不集中和注意持续时间短暂、活动过多和冲动为主要的临床表现。
- 症状可造成患者的学业困难的人际关系不良。
- 注意缺陷多动障碍主要与遗传、神经解剖学、神经生理学、神经生化、环境因素等有关。
- 注意缺陷症状主要表现为注意持续时间短暂，易受环境影响而转移。
- 活动过多和冲动体现在难以保持安静，不停活动，行为和情绪易冲动。
- 患者可出现学习困难，以及精细动作、协调动作能力较差神经发育异常。
- 对ADHD患者应采取因人而异的综合性方案进行干预。
- 药物治疗主要包括中枢兴奋剂和非中枢兴奋剂两类。
- 有说法认为，专注达可能会加重抽动症状，但如抽动已采取药物治疗且症状稳定，即不需要过于担心。

#### 强迫障碍

- 复杂抽动通常具有一定的“强迫性”。
- 强迫症即强迫障碍，是一种以反复出现的强迫性思维和/或强迫行为为主要临床特征的精神障碍。
- 强迫症会给患者带来严重痛苦，造成社会功能不同程度的受损。
- 强迫思维是指反复进入患者头脑中的侵入性的、非意愿的观念、意象或冲动，常伴有焦虑情绪。
- 强迫行为是指包括精神行为在内的非意愿进行的重复行为。
- 典型的强迫思维包括反复怀疑门窗是否关好、在某种场合出现自己不能控制的联想、不由自主地反复回忆以往经历等、脑内反复出现逼真的令其痛苦而不能控制的形象内容等。
- 典型的强迫行为包括反复确认门、煤气阀门、水龙头是否关好，因担心污染而反复洗手，反复计数，以及仪式性动作等。
- 强迫症的治疗遵循综合治疗、个体化治疗和全病程治疗的原则。
- 强迫症最常用的心理治疗方法是认知行为治疗（CBT）
- 药物治疗主要以5-羟色胺再摄取抑制剂为主。

### 难治性抽动障碍

- 难治性抽动障碍是近年来逐渐形成的新概念，目前国内外没有统一的观点。
- 国内普遍接受的观点如下
  - 重度抽动障碍经足量规范使用抗抽动药物（如氟哌啶醇、硫必利、阿立哌唑）等治疗1年以上
  - 仍无明显疗效而前言不愈者。
  - 临床上，经规范治疗1年以上YGTSS（耶鲁综合抽动严重程度评分）减分率仍不理想可考虑为难治性病例。
  - 同时强调需要综合治疗，除药物治疗外，还应包括心理干预等非药物治疗方式。

- 某些“难治性”病例源于伴有精神障碍、治疗不足、综合干预不充分等原因，并非难治性抽动障碍。
- 由于漏诊、误诊、伴有精神障碍、治疗不足、患者对药物缺乏耐受性或无法获得所有可用治疗方法等原因，临床上难治性抽动障碍的病例不断增加。
- 难治性抽动障碍应及时向儿童精神科转诊。

## 🚩为什么会患抽动障碍

- 关于抽动障碍的病因和发病机制，有许多疑点仍不明了。
- 目前的研究主要集中在抽动秽语综合征上。
- 但抽动障碍的发病机制被认为是免疫、遗传、心理和环境因素共同作用的结果。
- 素质-压力模型用以解释异常心理的机制，可以参考。
- 其认为，遗传因素作为素质，决定了能不能患病。
- 环境、社会、心理因素作为压力，共同决定了什么时候发病、发病的程度如何。

### 遗传因素

- 研究证实，遗传与抽动秽语综合征病因有关。
- 家系调查发现，10%~60%的患者存在阳性家族史。
- 双生子研究证实，单卵双生子的同病率（75%~90%）明显高于异卵双生子（20%）。
- 寄养研究显示，寄养亲属中抽动障碍的发病率显著低于血缘亲属。
- 研究还发现，抽动秽语综合征患者的亲属中慢性抽动障碍、强迫障碍、注意缺陷多动障碍的患病率显著增高。
- 目前大多数学者认为，抽动秽语综合征属于多基因遗传，且存在性别差异。
- 大量的研究尚未证实该病的特定基因及遗传方式，而且既往研究成果难以被进一步复制加以证实。

### 神经生物学

- 神经递质是神经系统中的信使，负责在神经细胞间传递信息。
- 皮质-丘脑-纹状体-丘脑-皮质通路（CSTC），被表明参与了抽动秽语综合征和它伴随的神经精神症状的表达。
- 在这条通路上，存在多种神经递质的失衡。
- 包括多巴胺能、5-羟色胺能、胆碱能、去甲肾上腺素能、组胺能和阿片类系统。
- 但多巴胺能系统可能在抽动秽语综合征中起到了最实质性的作用。

### 神经免疫学

- 对抽动障碍与感染后自身免疫的研究，较多的是与A组溶血性链球菌感染的关系。
- 有学者提出与链球菌感染有关的儿童自身免疫性神经精神疾病（PANDAS）假设。
- 临床上发现，感染发热可能使抽动症状加重。

### 心理因素

- 精神创伤、精神压力过大、情绪波动、疲劳与兴奋、过度惊吓等可诱发或加重抽动。
- 有研究认为，抽动障碍患者存在不同程度的个性异常，多为回避型或冲动型人格。人格因素可能在抽动障碍的发病中粗有一定的作用，可能为发病危险因素。
- 近年来有调查认为，家庭不良生活事件、教育方式不当、学校不良环境会加重抽动症状。

### 精神动力学模型与精神动力学治疗

- 关于其他心理治疗作为抽动障碍干预手段的内容，参见“寻求帮助”。
- 精神动力学是一个脱胎于精神分析的心理学派，其创始人是奥地利心理学家弗洛伊德。
- 精神动力学同时也是一种相对主流的心理治疗技术。
- 精神动力学的理论高度凝练抽象，动力学解释因人而异。
- 精神动力学内部理论模型很多，比如客体关系理论、自体心理学、主题间性视角等。
- 胡佩诚等《心理治疗》对心理动力学的阐述摘抄如下：

> 心理动力学的基本观点是假设无意识的心理活动可以影响有意识的思想、情感和行为。
> 心理动力学治疗是指以心理动力学的观点为基础的心理治疗方法。

- 《心理动力学疗法：临床实用手册》对心理动力学心理治疗的定义如下：

> 心理动力学治疗是一种谈话治疗，它的基本观点是人们受无意识思想的影响和驱动。它的目标是在与治疗师的关系背景下，通过帮助人们了解自己的心灵如何工作或直接支持他们的行动，进而帮助他们改变惯有的思维和行为方式。

- 精神动力学认为，任何一种症状都有意义。症状是潜意识（潜意识、无意识是同义词）冲突的妥协形成。
- 防御机制是心理在内外压力和情绪冲突下做出无意识和自动化反应的方式。
- 躯体化，是一种防御机制，是指心理冲突，可能在压抑下，通过躯体症状的形式表达了出来。
- 精神动力学重视发掘童年经历与症状的关系。
- 抽动不仅是神经递质的失衡，还是心理试图表达、防御或控制某种无法言说的内心内容的躯体化形式。

#### 自我心理学

- 抽动可以理解成一种在冲动与防御之间达成的动态平衡。

##### 被压抑的冲动

- ##### 抽动可能包含着某种与被禁止的冲动，比如攻击性、性本能、愤怒。

- 秽语可以理解为被压抑的攻击性语言以失控的形式冲出防线，又以一种扭曲、重复、非交流性的形式出现，使其不至于承担人际冲突的后果。

##### 防御的扭曲表达

- 抽动本是也是一种防御，将心理冲突转换到躯体运动。
- 借此，个体将无法承受的心理内容外化到身体层面。
- 这是一个从心理退行到躯体的过程。

##### 控制与失控

- 抽动是一种“为了控制而失控”的症状。
- 患者短暂抑制抽动，但这种抑制导致内在张力的不断升高，最终以更强烈的形式爆发。
- 动力学可能假设患者存在自我惩罚的症结，个体试图控制某些不可接受的情感或冲动。
- 这种控制本身是紧张的来源，最终导致症状的爆发。

#### 客体关系视角

- 根据客体关系理论，抽动不仅是内在驱力的衍生物，更是早期关系模式在身体层面的“活现”。

##### 被内化的冲突性客体关系

- 如果个体早年生活在一个功能失调的家庭中，例如，养育者一方面要求绝对服从，另一方面又通过非言语方式表达愤怒，孩子可能内化一种“既想反抗又必须服从”的内在关系模式。
- 抽动的症状形式有一个肌肉群与另一个肌肉群同时收缩的特点，可以理解为这种内在僵持的躯体化呈现。

##### 身体作为战场

- 当言语无法承载关系中的张力：
- 例如家庭中某些话题被禁止，某些情感不被承认，身体可能成为替代的战场。

##### 攻击转向自身

- 如果个体无法安全对外表达情感，攻击性可能转向自身。
- 抽动中的自我击打、碰撞，或者仅仅是肌肉痉挛，都可能包含着这种转向自身的攻击。

#### 自体心理学观点

- 从科胡特的自体心理学出发，抽动可以理解为自体受损后的碎片化表达。

##### 对镜映失败的躯体反应

- 当个体的情感、需求和自发性表达长期得不到养育者的关注和确认，他可能逐步失去对自己内在状态的感知能力。
- 那些没有被“镜映”的情感，无法被整合进自体结构，最终可能以躯体症状的形式“泄露”出来。

##### 孪生需求的挫败

- 抽动障碍患者常感到自己“怪异”、“与他人不同“。
- 这种孤独感可能来自于没有体验到“有人像我一样”的基本连接感。
- 抽动有可能是对这种孤独的表达。
- 也是一种试图吸引他人注意，寻找链接的扭曲性努力。

##### 对解体的防御

- 在某些压力情境下，个体可能体验到一种弥漫的解离感与自体解体的恐惧。
- 抽动作为一种强烈的、可感知的身体刺激，可以瞬间将注意力拉回身体。
- 进而确认“我还存在”、“我有边界”。
- 它是一种用身体感来对抗自体解体感的措施。

#### 心智化理论

- 心智化是理解、觉察自身和他人心理状态的能力。

##### 心智化失败

- 在高度情感唤起的状态下，个体的心智化能力可能崩溃，退行到“心理等同”的模式。
- 即内心感受直接等于外部显示，无法被反思。
- 抽动可能是心智化失败的产物。
- 无法被“作为心理状态来思考”。

##### α功能的失效

- 比昂认为，α功能是将原始的β元素转换为可被思考的心理元素的能力。
- 抽动可以被理解为β元素的直接躯体释放。
- 那些不被涵容、无法被转换的心理碎片，以最原始的方式从身体爆发。

##### 前心智化的躯体表现

- 发展心理学中，婴儿通过躯体动作来表达心理需求。
- 当养育者通过“标记性镜映”，婴儿才能逐渐学会用心理语言代替躯体语言。
- 抽动可能暗示了这个转化过程在某个关键节点的受阻。
- 个体在某些情境下退行到前心智化水平的躯体表达模式。

#### 总结

- 精神动力学模型并不与神经科学解释相对立，而是不同维度的互补关系。
- 在心理动力学的视角中，抽动不是要被“消灭”的东西，而是一个身体“替”心理发出的信号。
- 心理动力学理论认为，治疗的一般作用原理则主要包括治疗同盟的作用。
- 而治疗的特殊作用原理主要包括将无意识内容意识化出来等。
- 动力学心理治疗师的作用是，做一个稳定的容器，帮助个体把身体语言“翻译”回心理语言，探索抽动症状背后的意义，支持未完成的发展任务。
- 抽动的减轻可能是在治疗中工作，发现心理冲突、心智化能力得到增强的副产物。

## 🩺寻求帮助

### 就诊

#### 基础原则

- 抽动障碍是可干预、需治疗的神经发育障碍。
- 不要“讳疾忌医”，抽动障碍患者要到正规医疗机构就诊。
- 儿童精神心理科医生是诊治神经发育障碍和儿童期精神障碍的专科医生。
- 儿童精神科医生可以提供对抽动障碍比较完整的评估和综合治疗方案。
- 在就诊时，要完整提供病史，展示之前用过的治疗方式，并与医生充分沟通。
- 抗抽动药物（包括精神药物）经过长期的安全性检验，不要妖魔化药物治疗。
- 心理治疗是国际上推荐的抽动障碍一线治疗，不要轻视心理治疗。
- 抽动障碍有相对完整的干预方案。
- 大多数患者经过规范系统的综合治疗（包括药物治疗和心理治疗等），其临床症状和生活质量会得到相应的改善。

#### 寻找医生

- 我国当前儿童精神科医生总数少于大熊猫。
- 如果当地并无儿童精神科医生出诊，参见“替代方案”。
- 儿童精神科医生一般会出现在当地精神专科医院医院的“儿少门诊“，可能也会在其他门诊处出诊。
- 部分大型儿童医院也会设立心理科。
- 先查询当地相关医生的出诊地点和时间，规划时间，在手机上预约挂号。
- 未成年人到精神心理科就诊必需监护人陪同，否则无法开药治疗。
- 大部分精神专科医院的精神科普通/专家诊和心理咨询与治疗门诊是分开的。
- 精神科门诊负责问诊、评估、开药，并可能会建议患者结合心理治疗。
- 心理治疗号需要单独预约，或在医生处开单（如北大六院）。
- 抽动障碍相关医生简介和体验，链接（待开发）。

##### 替代方案

- 如当地偏远而无儿童精神科医生出诊，或者约号困难，经验上可以按以下顺序尝试：
  - 精神专科医院 / 大型综合医院心理科，医生介绍写有“儿童行为情绪问题”“多动症抽动症”等关键字，或电话联系确认该医生可接诊抽动障碍患者。
  - 儿童医院发育行为儿科、神经内科门诊。
  - 儿童医院儿童保健科。

- 不推荐或大概率不接诊：
  - 综合医院神经内科。

- 超一线城市等大城市可考虑：
  - 综合医院神经内科运动障碍门诊。

- 虽然由于历史因素和现实状况，上述科室的医生也在从事抽动障碍的诊疗工作，但他们：
  - 面对难治性的抽动障碍患者，缺少诊治经验。
  - 缺乏对抽动障碍精神共病的认识，难以进行系统的心理评估。
  - 没经过正式的综合干预、心理治疗培训，视角局限，治疗理念仅限于开药。
  - 没经历过完整的精神药理学训练，不熟识精神药物的使用，药物治疗规范性欠佳。
  - 可能过于谨小慎微，反复建议进行躯体检查，错失适当的干预时机，带来经济负担。


#### 到院就医

##### 精神专科医院

- 务必仔细阅读挂号页面的要求信息。

- 有一部分精神专科医院要求首诊建档登记，或者补交精神检查费+病案建档费，例如大连专科医院是60元，首诊需到护士台填表。

- 大部分医院就诊前先要签到。

- 医生会先采集病史，询问例如起病时间、症状变化等问题。

- 医生不是大家长，通常不会否认你的痛苦，更不会主动抨击你的想法，这是精神科医生的基本素养。

- 你只需要在有限的时间内尽可能完整、扼要地表达你的所有困扰，对医生的问题据实回答即可。

- 建议在时间允许的情况下，说出自己的想法或者担心，和医生充分沟通。

- 部分医生可能会选择给抽动障碍患者，尤其是成年的和有共患病可能的患者，开一些心理测查（各种评估量表），例如SCL-90，来更好地了解你的整体情况，以及症状严重程度。

- 量表单独收费，一般每张50~100元左右。

- 抽动障碍的诊断有一定的排除性标准，因此医生可能会开一些躯体检查来排除其他疾病。但由于症状的典型性，有些医生可能不会这样。
  - 头CT或头MRI有助于排除脑器质性疾病。
  - 脑电图有助于排除痫性发作。
  - 血浆铜蓝蛋白有助于排除肝豆状核变性。
  - 抗“O”等有助于排除小舞蹈病（风湿性舞蹈病）。
  - 心电图、血常规和血生化（肝肾功能等）可能作为药物治疗前的评估。

- 如有需要，可打印病历、诊断证明。

- 如建议服药治疗，医生会嘱咐药品用法，抗抽动药物一般不会带来经济负担。

- 少数情况下，如精神共病严重，医生可能建议住院治疗。
  - 开放式病房比封闭式病房更为人性化，但可能难以预约。
  - 询问好住院要求，征询患者本人意见。
  - 如不想住院，可直接要求门诊治疗。


#### 转诊

- 对难治性抽动障碍，可到北京等地的儿童精神科专家处就诊。

- 北京采用114平台统一预约挂号，可在微信小程序上操作。

- 可在挂号页面、医院官网处查到医生的专长。

- 如无经济困难，可预约特需门诊，以节省时间。

- 举例来说
  - 首都医科大学附属北京安定医院是北京一所三级甲等精神专科医院，设有儿童青少年精神科。其中郑毅儿童专家团队比较著名，陈旭、何凡及团队其他骨干均有特需门诊出诊，价格在300~500元不等。可在114平台预约。


### 综合干预

#### 总述

- 如前所述，综合干预在抽动障碍的治疗中非常重要。

- 近十年来，抽动障碍领域有很多重要变化。
  - 从前在抽动障碍的治疗中，被认为”无效甚至有害“的心理干预，被欧洲抽动秽语综合征研究学会（ESSTS）、美国神经病学学会（AAN）、加拿大儿科学会等国际指南推荐为抽动障碍的一线治疗方案，完成了“辅助”到“基石“的转身。
  - 抽动综合行为干预（CBIT）成为研究最深入、证据最充分的心理干预方法。
  - 其他心理治疗方法，如认知行为治疗（CBT，我国治疗师基数大）也有积极作用。
  - 非典型抗精神病药已存在典型药物，成为中重度抽动的一线选择。
  - 其中，阿立哌唑的地位显著上升，获FDA批准抽动障碍适应证。
  - 新药Ecopipam的3期临床试验结果公布，是十年来最具突破性的药物研发进展。

- 认知行为治疗、精神动力学治疗等是抽动障碍的精神共病，例如强迫症、抑郁障碍的有效疗法。

#### 心理治疗

- 心理动力学治疗的相关内容参见“精神动力学模型”。
- 胡佩诚等《心理治疗》中对心理治疗的定义如下：

> 心理治疗或称精神治疗，是以一定的理论体系为指导，以良好的医患关系为桥梁，应用心理学的方法，影响或改变患者的感受、认知、情绪及行为，调节个体与环境之间的平衡，从而达到治疗目的。

- 心理治疗与心理咨询有着相对的界限。
- 心理治疗的工作对象可称患者，而心理咨询的工作对象主要是在适应和发展上发生困难的普通人。
- 我国《精神卫生法》规定，心理治疗只能在精神卫生机构中开展，心理咨询师不能诊断或治疗精神障碍。
- 精神科医师经过心理治疗培训后可从事心理治疗工作，我国也有专职的心理治疗师。
- 信任、保密、计划、针对性、综合、灵活、中立、回避是心理治疗的几项基本原则。
- 根据学派理论等，心理治疗可分为心理动力学治疗、认知行为治疗、人本主义心理治疗等方式。
- 技术上对不同理论的整合是心理治疗的一种发展趋势。
- 抽动症状与情绪密切相关，而心理治疗有助于情绪管理能力的提高——这是一条最简单的作用原理。
- 研究证明，心理治疗对抽动症状及抽动障碍患者的生活质量的改善都有积极作用。
- 我国精神专科医院心理治疗费用水平不等，按职称也有异，在300~1000元/次左右。
- 按照医院规定，一次心理治疗通常在45分钟左右。

##### 抽动综合行为干预

- 抽动综合行为干预（CBIT）在国际上是研究最深入的针对抽动障碍的一线治疗。
- CBIT是在行为逆转训练的基础上进一步整合和发展而来的。
- 但是，CBIT在国内尚属起步阶段，只有极少数医院开展了相应的治疗。
- 虽然CBIT资源的可及性非常不足，但其他心理治疗仍然有益于抽动障碍的干预。

##### 认知行为治疗

- 认知行为治疗（CBT）是一种以认知行为模型建立的一种目前问题取向的、短程的、结构式的心理治疗。
- 认知行为治疗的适应证十分广泛。
- 按照Beck的观点，其基本方法几乎适用于任何精神障碍。
- 其已获得了大量的研究关注并取得了相当多的证据支持。
- 认知行为治疗模型涉及三个基本概念：认知、情绪和行为。
- 认知是指人对事件或者情境的态度、看法、信念等。
- 情绪是人的内心体验；行为是人的外在表现。
- 认知行为治疗的基本模型模型的核心是认知的中介作用。
- 其认为人的认知活动可以被监测和改变的。
- 认知行为治疗的治疗师会采用苏格拉底式提问、行动计划等方式改变个体歪曲的认知和非适应性行为。

#### 药物治疗

##### 总述

- 对于中重度抽动障碍患者，通常需要药物治疗。

- 规范应用的抗抽动药物（包括精神药物）安全且可控。

- 其不会成瘾，并且有助于中重度抽动障碍患者症状的改善，可以改善其对生活的困扰。

- 抗抽动药物大多是off-label，即说明书外用药，这并不代表将他们用于抽动是未经考证的或者不安全的。

- 药物治疗从小剂量开始，逐渐滴定到有效剂量。

- 如出现不良反应，不代表治疗失败，应和医师沟通，考虑观察、换药或者其他处理方法。

- 药物治疗方案一般分为急性期、巩固期和维持期。
  - 急性期治疗的目的是积极控制患者的症状，缩短病程，改善预后。急性期疗程长短不一。
  - 巩固期，即巩固急性期疗效，防止复燃。病情基本控制后，需继续维持急性期治疗至少1~3个月，予以巩固其疗效。
  - 维持期的目的是防止复发，提高生活质量。即巩固治疗后病情控制良好，仍需维持治疗6~12个月。维持剂量一般为治疗剂量的1/2-2/3，或根据病情缓慢减量到合适剂量。

- 联合用药是指，当使用一种药物仅能使部分症状缓解或共患病明显时可考虑联合用药。

- 难治性抽动障碍大多需要联合治疗。

- 停药是指，经过维持治疗阶段后，若病情完全控制，可以考虑逐渐减停药物。

- 减量期至少1~3个月，缓慢减量在于防止抽动症状反弹加重。

- 若确定复发，需再恢复到原有药物或换用其他药物治疗。

- 药物治疗处方应由相应医生做出。

#### 神经调控治疗及其他技术

##### 神经调控治疗

###### 总述

- 抽动障碍遵循综合干预的原则，在中重度患者中单独应用神经调控治疗并不是科学的。
- 非药物性疗法如神经调控治疗正在受国内外学者的关注。
- 一个粗浅的定义是，神经调控治疗在神经精神疾病领域，是指一类通过物理（电、磁、光、超声）或化学手段，定向调节中枢或外周神经系统特定靶点的神经元活动，从而改善神经精神症状的侵入性或非侵入性治疗技术。
- 神经调控治疗包括脑电生物反馈、经颅磁刺激（TMS）、经颅微电流刺激（CES）、深部脑刺激（DBS）等。
- 多项多中心、双盲、随机对照实验显示了治疗的有效性。然而，其仍局限于短期、小样本研究，治疗的长期预后尚需进一步的随访观察。
- 总结来看，神经调控治疗定位明确，但证据水平弱于心理治疗或药物治疗。
- 即使是无创性手段，不同医生对神经调控治疗的积极性也不同，观点亦有可能有差异。
- 部分精神专科医院设有神经调控中心，综合医院神经内科也有可能设置有神经调控治疗室。
- “家庭治疗仪”很有可能并不规范，其技术参数和医院所设置的治疗设备往往大相径庭，偏信这些广告，有可能导致经济损失，延误治疗。

###### 无创性神经调控治疗

- TMS、CES具有无创性和安全性的优势。
- TMS相对来说，推荐等级和证据水平在无创性神经调控治疗中比较高，但证据仍不足。

###### 深部脑刺激

- 深部脑刺激（DBS）是有创治疗，指通过应用立体定向手术技术将刺激电极植入特定脑区，通过电脉冲抑制其异常的震荡活动，达到治疗抽动障碍的目的。

- DBS严格限定用于年长儿童或成人难治性抽动障碍的治疗。标准参考“难治性抽动障碍”。

- 国外认为，应用DBS，应有多学科团队再根据患儿情况最大限度使用多种药物及行为疗法，排除继发性及心理诱因后才可考虑DBS治疗。

- 最新欧洲临床指南建议
  - 符合DSM-5或ICD-10的诊断标准。
  - 必须排除功能性“抽动样”运动障碍。
  - 主要治疗目标是减少抽动而非改善并发症。
  - 抽动秽语综合征对患者造成严重的功能损害且药物治疗效果欠佳。
  - 在治疗前必须考虑治疗效果和耐受性。
  - 由于抽动秽语综合征患者在青春期后或成年后有一定的自愈性，不推荐DBS最低手术年龄。
  - 应有一个专门的包括精神科、神经科、心理科在内的团队对患者进行全面评估与管理。
  - 应遵循特定的方案且对患者治疗前后做系统评估。

- 目前国内只有比较顶尖的神经外科医院，例如北京天坛、北京宣武等开展了针对抽动障碍的深部脑刺激技术。

##### 外科手术

- 外科手术可能是难治性抽动障碍的最后选择。
- 是指对皮质、扣带回、丘脑或者小脑区域进行一系列的神经外科干涉。
- 需要指出，抽动障碍的外科手术治疗必须严格选择手术指征。
- 患者术后会面临许多挑战，包括术后可能的并发症、后遗症和不良反应。
- 目前关于手术治疗难治性抽动秽语综合征的年龄仍存在争议，国外一些专家把手术年龄定为＞18岁。
- 目前没有证据证明一种手术较另一种手术疗法高明。
- 抽动障碍的神经外科治疗只能被看做一个实验性的技术，尚需进一步扩大样本验证。
- 开展外科手术的医院国内极少。
- 应避免前往经验不多、团队结构单一的医院治疗。
- 如果经药物治疗、心理治疗、神经调控治疗等确认无效，也应到北京天坛、北京宣武等顶尖神经外科咨询。

### 中医药

#### 总述

- 其实儿童抽动障碍在中医范畴中并无完全对应的病名。
- 根据患儿其眨眼、皱眉、努嘴、摇头、肢体抽动、清嗓、发声等核心症状，多归属于“慢惊风”“瘛疭”“肝风”“筋惕肉瞤”等范畴。
- 在文献中也有医家将其纳入“郁证”“梅核气”等相关病症讨论。
- 中医认为，本病的发生与小儿生理病理特点密切相关。
- 以肝失疏泄、肝风内动为核心病机。
- 同时涉及脾、肾、心等脏腑功能失调。
- 且情志失调、饮食失宜、外感邪毒等为重要诱发因素。
- 辨证论治是中医治疗抽动障碍的基本原则。
- 即使是中成药，也应在四诊合参、中医辨证后应用，因人而异。
- 治疗以平肝息风为基本大法。
- 兼顾健脾化痰、滋肾养肝、清心安神、疏肝解郁等。
- 同时注重调护情志、规范饮食，实现辨证论治与整体调理的结合。
- 如需寻求中医治疗，可考虑到中医院中医儿科、中医脑病科等就诊。
- 中药处方应由医师做出。

## 🍃生活和养育

### 抱持性环境

- 当儿童被诊断为抽动障碍，或者出现抽动样症状，你可能已经查了很多资料、跑了很多医院、试了很多种方法。
- 你可能在无数个夜晚问过自己：我到底该怎么做？是严格管束，还是完全放任？提醒他控制，还是装作没看见？
- 这里将从一个你可能没听说过的角度——“抱持性环境”来谈一谈你可以为孩子做些什么。
- 这个概念来自英国儿科医生、精神分析家温尼科特。
- 抱持性环境的英语就是holding environment。
- 我国精神分析学者曾奇峰说，抱持就是“hold住”。

#### 什么是抱持性环境

- 讲个比喻。

- 想象一个刚学走路的孩子。

- 他摇摇晃晃地迈出第一步，随时可能摔倒。

- 但他敢往前走。

- 是因为他知道，即使摔倒了，也会有一双温暖的手接住他。

- 他不会摔得头破血流，不会被嘲笑“怎么又摔了”，不会被谁说“自己爬起来，不许哭”。

- 这个“有人接着、有地托着”的感觉，就是抱持性环境。

- 抱持性环境不是

- - 把孩子包起来、什么都不让他做。
    - 孩子需要面对挑战，但当他面对挑战时，他直到身后有一个安全基地可以随时回来。

  - 把孩子扔出去、让他自生自灭。
    - 孩子需要边界、有规则，但边界的设立是出于关心，而不是出于愤怒和控制。

  - 只关注抽动
    - 你是在和一个完整的人相处，而不是和一个“有抽动”的人相处。

  - 永不发火
    - 父母也是人，也有情绪。
    - 抱持性环境不是要求完美，而是当父母有情绪后，能够修复、能够解释、能够让孩子知道：
    - “刚才是我自己的问题，不是你的错。”

- 抱持性环境是
  - 在孩子需要支持的时候，有支持。
  - 在孩子需要空间的时候，有空间。
  - 在孩子害怕的时候，有一个安全的地方可以回来。


#### 为什么是抱持性环境

- 从心理动力学的角度看，抽动可以被理解为一种“当语言时效时，躯体替心灵说话”的现象。

- 以下环境会让孩子“语言失效”，被迫用身体“直接行动”。
  - 当孩子表达情绪时，被说“不许哭”、“有什么好生气的”。
  - 当孩子说出自己的想法时，被驳回“你懂什么”、“听我的”。
  - 当孩子尝试自主时，被过度控制“你必须按照我说的做”。
  - 当孩子表现出不一样时，被评判“你怎么这么怪”。

- 因此，对抽动障碍的孩子来说，这个抱持特别重要。

- 因为抽动本身，就是一个内在的“不稳定因素”。

- 它让孩子感到失控、感到羞耻、感到自己和被人不一样。

- 如果外在环境也是紧张的、评判性的、不可预测的。
  - 孩子的内心会更加混乱。
  - 抽动会更加严重。

- 如果外在环境是稳定的、接纳的、可预测的。
  - 孩子就能慢慢学会和自己的抽动相处。
  - 从而利于心理成长和抽动的减轻。

- 抱持性环境做到，是创造一个让孩子可以用语言，而不是用身体来表达自己的空间。

#### 六个具体策略

##### 创造一个“情绪可以说话”的空间

- 这是最核心的一条。

- 抽动是在替身体说话，那我们就帮孩子用自己的语言器官说话。

- 当孩子有情绪时，试着帮他把感受“翻译出来”
  - “刚才你好像很烦躁，是不是有什么事让你不舒服？”
  - “妈妈注意到你在做作业时抽动变多了，是不是觉得压力大？”
  - “你刚才那个动作，好像是在生气？我猜对了吗？”

- 不需要猜对。

- 只需要让孩子知道：你的感受是值得被关注的，你的情绪是可以被讨论的，你可以用语言，而不是身体，来让别人理解你。

##### 对抽动本身的态度：观察、不评判

- 这是最难，也最重要的一步。

- 当孩子抽动时，我们的本能反应可能是：紧张、焦虑、想提醒他控制、想问他“你怎么又这样了”。

- 但这些反应本身，会增加孩子的压力。

- 他会觉得“我的抽动让妈妈担心了”、“我又让爸爸不高兴了”，于是更紧张，抽动更多。

- 可以试着这样做。
  - 看见抽动时，深呼吸一下，提醒自己：“这是他的症状，不是他的错。”
  - 如果必须回应，可以用平静的语气说一句：“我看到你刚才动了一下，没事，继续做你的事吧。”
  - 或者，更简单的——就像没看见一样，继续该干嘛干嘛。

- 这不代表冷漠，而是代表：抽动在这个家里，不是一件需要大惊小怪的事。

##### 区分“症状”和“行为”

- 抽动是不由自主的，但有些伴随行为可能是故意的、有目的的。家长需要学会区分：
  - 如果是不由自主的抽动——不批评、不纠正、不提醒。
  - 如果是因为抽动影响了别人（比如不小心打到人）——温和地处理后果，但不把“抽动”和“坏孩子”挂钩。
  - 如果孩子用抽动作为借口逃避某些事（比如“我不想去上学因为我怕抽动”）——需要和孩子一起面对背后的恐惧，而不是简单地允许或禁止。

- 区分的关键是：理解孩子，而不是审判孩子。

##### 帮助孩子理解自己，而不是控制自己

- 您可能会问：“怎么才能让他少抽动？”这个问题本身可能就是一个问题——因为它把焦点放在了“控制症状”上。

- 更好的问题可能是：
  - “怎么帮他理解自己的抽动，让它不那么困扰他？”

- 您可以和孩子一起探索：
  - “你觉得什么情况下抽动会变多？紧张的时候？无聊的时候？”
  - “抽动之前，身体有什么感觉？有没有办法在那个感觉刚出现的时候做点什么？”
  - “如果抽动能说话，你觉得它会说什么？”

- 这些问题不是治疗，而是在传递一个信息：
  - 你的身体是你的，你的感受是重要的。
  - 我们一起来理解它，而不是对抗它。


##### 调整家庭中的“压力源”

- 抽动和孩子当下的情绪状态高度相关。而孩子的情绪，往往和家庭氛围密切相关。

- 可以试着观察一下
  - 家里的规则是不是太多、太严？
  - 父母的期待是不是太高、太急？
  - 夫妻之间是不是经常冲突？孩子是不是成了“夹心饼”？
  - 有没有给孩子足够的自主空间？哪怕是很小的选择（穿什么、吃什么、怎么安排时间）？

- 调整这些，不是为了“治好”抽动。

- 而是为了创造一个更健康的环境。

- 即使抽动还在，孩子也能在这个环境里健康成长。

##### 家长自己的“抱持”也很重要

最后，也是同样重要的一条：***\*家长也需要被“抱持”\****。

- 养育一个抽动障碍的孩子，是一件很辛苦的事。

- 你可能焦虑、无助、疲惫，可能自责“是不是我做错了什么”。

- 可能面对外界的误解和指指点点。

- 如果你自己的情绪没有地方安放，你也很难给孩子提供一个稳定的环境。所以：
  - 给自己找一个倾诉的地方——伴侣、朋友、支持团体、专业人士（例如心理咨询师）。
  - 允许自己有情绪，但不让情绪控制你的行为。

- 你已经是孩子最重要的“容器”，但这个“容器”也需要被装满。

#### 一个抱持性环境的家庭日常

- 最后，想象一个这样的场景。

> 晚上，孩子在做作业。他的抽动开始变多——耸肩、清嗓子。妈妈在旁边看书，看到了，但没说话。
> 过了一会儿，孩子突然说：“妈妈，我今天在学校好烦。”
> 妈妈放下书，转过来看着他：“哦？发生什么事了？”
> 孩子开始讲学校的事——和同学闹矛盾、被老师批评、感觉自己像个怪人。说着说着，他哭了。妈妈没说话，只是把手放在他肩上。
> 哭完之后，孩子的抽动明显少了。他说：“妈妈，我好多了。”
> 妈妈说：“嗯，那些事确实挺烦的。下次再有这种事，随时可以跟我说。”
> 然后继续看书，孩子继续写作业。抽动还在，但已经不重要了。

- 这就是一个“抱持性环境”的日常。
  - 不是没有症状。
  - 不是没有情绪。
  - 而是症状和情绪都有地方可以安放。


#### 结语

- 抽动障碍的治疗，医学和心理是两个层面。

- 作为家长，你们能够提供的，是那个让改变发生的、重要的“土壤”。

- 一个稳定的、安全的、接纳性的环境。
  - 在这个环境里，孩子不需要用身体去替心灵说话，因为他已经可以用自己的语言被听见。
  - 在这个环境里，抽动可能还在，但它不再是孩子羞耻的秘密，也不再是家庭焦虑的中心。
  - 它只是孩子的一部分，而不是孩子的全部。

- 你们正是这个环境最重要的守护者。

### 日常生活

- 每个人都是独特的。
- 抽动障碍是你的一部分，但不是你的全部。
- 虽然患有抽动障碍，但这并不消减你的价值。

##### 向谁暴露“我是抽动障碍的患者”？

- 抽动障碍的诊断是你的个人健康信息，你没有义务告诉任何人。
- 是否暴露、向谁暴露、何时暴露、暴露多少，完全属于你的权利。
- 这不是诚实与否的道理问题的，而是对隐私边界的捍卫。
- 你可以根据情境和内心感受，自由选择如何暴露。
- 你不需要一次性告诉所有人，也不需要永远不告诉任何人。
- 可以从最安全的人开始。
